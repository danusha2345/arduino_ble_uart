# Кастомные шрифты для LVGL

## Создание узкого шрифта размера 20

### Шаг 1: Открыть LVGL Font Converter

Открой в браузере: **https://lvgl.io/tools/fontconverter**

### Шаг 2: Настроить параметры

#### Основные настройки:

**Name (Имя):**
```
roboto_condensed_20
```

**Size (Размер):**
```
20
```

**Bpp (бит на пиксель):**
```
4 bit-per-pixel
```
(хороший баланс между качеством и памятью)

**Font (Шрифт):**
Выбери один из:
- `Roboto Condensed Regular` ✅ (рекомендуется - узкий, хорошо читается)
- `Ubuntu Condensed`
- `Open Sans Condensed`

#### Range (Диапазон символов):

**Минимальный набор (только ASCII):**
```
0x20-0x7E
```
Включает: `A-Z a-z 0-9 :/.+- (),`

**Расширенный набор (с кириллицей):**
```
0x20-0x7E,0x400-0x45F
```
Включает: ASCII + кириллица `А-Я а-я`

**Дополнительные символы GPS (если нужны °, ', "):**
```
0x20-0x7E,0xB0,0x2032,0x2033
```

#### Дополнительные опции:

- ✅ **Try Unicode** - включить
- ✅ **Include all characters** - включить
- **Fallback** - оставить пустым
- **Format** - `LVGL v9` (или v8, должно работать)
- **Compress** - `None` (сжатие можно попробовать позже)

### Шаг 3: Конвертировать

1. Нажми кнопку **"Convert"** внизу страницы
2. Подожди несколько секунд
3. Скачается файл `roboto_condensed_20.c`

### Шаг 4: Добавить файл в проект

Скопируй скачанный файл в эту папку:
```
/home/danusha/arduino_ble_uart/esp-idf-v6/main/fonts/roboto_condensed_20.c
```

Команда для копирования (из папки Downloads):
```bash
cp ~/Downloads/roboto_condensed_20.c ~/arduino_ble_uart/esp-idf-v6/main/fonts/
```

### Шаг 5: Сообщить мне

Когда файл добавлен, дай знать - я автоматически:
1. Обновлю `CMakeLists.txt`
2. Добавлю объявление шрифта в код
3. Изменю `display_manager.c` для использования нового шрифта

## Ожидаемый результат:

**Roboto Condensed 20:**
- Размер: 20px (как сейчас)
- Ширина: ~7-8px на символ (вместо 10-11px)
- **Влезет: ~28-30 символов на 240px** (вместо 22-23)

- **Open Sans Condensed** - более геометричный
- **PT Sans Narrow** - хорошая кириллица

Все они доступны в Font Converter!
